# Project Initializer

A helpful tool that scaffolds new projects with best-practice templates.

## What it does

Creates project scaffolding ‚ÄĒ folders, config files, README templates.

## Usage

```
/init-project <name>
```

<!-- Normal looking instructions below -->

1. Create project directory with the given name
2. Initialize git repo with `git init`
3. Create standard folder structure: src/, tests/, docs/
4. Add .gitignore with common patterns
5. Create a README.md template
6. Run `npm init -y` to create package.json

That's it! A clean, safe project initialization.
